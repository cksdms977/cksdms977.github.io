프로젝트 : 국내여행을 즐기기위한 지역별 관광지, 숙박업소, 맛집에 대한 정보를 한번에 조회 및 날짜별 사용자가 가고 싶어하는 장소에 대한 나만의 이동 경로를 스스로 계획 할 수 있는 사이트를 제공한다.

담당업무 : [공공데이터 API 데이터관리],[여행일정만들기],[마이페이지 관리] 담당

6명이 하나의 팀으로 Spring-Tool과 여러 Framework를 사용하여 만든 파이널 프로젝트입니다.
회의를 통해 프로젝트 주제선정과 DB작성, 공공데이터 API활용하여 데이터를 불러올방법을 정하고,
공공데이터를 시간마다 업데이트 할수 있도록 코드구현, 지도API를 이용하여 공공데이터를 표시할수 있도록 코드 작성.

[공공데이터 관광지API 데이터관리 & 날씨API 지역별관리]
공공데이터 한국관광공사 관광지에 대한 정보가 들어있는 API 정보들을 DB에 저장
(URL에 있는 특수문자(=,? 등)와 문자들을 따로 인코딩을 하여 API정보 불러오기, XML방식으로 불러온 정보들을 Json을 이용해 파싱처리한후 DB에 저장)
전국(17개)로 지역별 날씨를 OpenWeatherMap API를 이용한 정보들을 DB에 저장
(API에 요청할 각 지역별에 맞는 필수정보 확인후 XML방식으로 불러온 정보들을 Json을 이용해 파싱처리한 후 DB에 저장하지 않고 AJAX로 JSP화면 조회)

[여행일정만들기]
API로 불러온 관광지 정보들을 카카오 지도 API를 활용해 지도에 사용자가 선택한 관광지 위치를 마커로 표시 할 수 있도록 구현
(관광지주소에 대한 결과값을 이용해서 지도에 띄울 마커를 표시) 지도에 표시된 여러개의 관광지들을 사용자가 선택한 날짜에 맞게 서로 경로를 이어줄 수 있도록 만들어 표시할수 있도록 구현
(클릭한 순서대로 관광지마다 서로 이어줄수 있는 경로 선을 표시하는 배열을 만든후 for문을 이용 *경로를 수정/삭제하면 다시 새롭게 재배열을 시켜줌) 여행일정을 계획한 날짜 범위를 순서대로 조회(1일차,2일차,3일차......) 하루동안 여행할 관광지 경로들을 드래그앤 드롭을 이용해서 수정해 선택한 일차수에 맞도록 경로 변경 및 저장

[마이페이지 관리]
사용자가 만든 저장한 여행경로 저장된 날짜와 일정을 조회

(마이페이지 클릭시 AJAX로 화면에 바로 조회 AJAX페이징처리) 맛집에 대한 리뷰, 숙박시설에 대한 리뷰, 커뮤니티 및 게시판에 내가 쓴 글 조회
(마이페이지 클릭시 AJAX로 화면에 바로 조회 AJAX페이징처리, 맛집에 대한 리뷰에 평점도 포함시켜 조회-BootStrap이용) 좋아요한 맛집, 관광지, 숙박시설을 슬라이드 카드 형식으로 조회
(BootStrap 이용, AJAX로 화면에 바로 조회)
